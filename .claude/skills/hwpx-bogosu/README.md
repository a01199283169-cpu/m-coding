# hwpx-bogosu

한글 문서(hwpx) 보고서 자동 생성기

## 개요

두 가지 표준 보고서 양식을 기반으로 한글(hwpx) 문서를 자동으로 생성합니다.

- **양식1**: 정식 보고서 기본 양식 (상세형) - 공공기관/대학교 공식 문서용
- **양식2**: 보고서 요약 양식 (간략형) - 신사업 보고서/내부 보고서용

## 설치 및 요구사항

### Python 버전
- Python 3.6 이상

### 필수 파일
```
hwpx-Bogosu/
├── hwpx_generator.py          # 메인 생성기
├── assets/                     # 템플릿 파일
│   ├── (샘플양식1) 보고서 기본 양식.hwpx
│   └── (샘플양식2) 보고서 기반 양식(요약).hwpx
├── output/                     # 생성된 파일 저장
├── examples.sh                 # 사용 예시 스크립트
├── hwpx-bogosu.skill.md       # 상세 문서
└── README.md                   # 이 파일
```

### 라이브러리
Python 표준 라이브러리만 사용하므로 별도 설치 불필요:
- zipfile
- argparse
- datetime
- pathlib

## 사용법

### 기본 문법

```bash
python3 hwpx_generator.py --template [1|2] --title "문서 제목" [옵션...]
```

### 필수 옵션

| 옵션 | 설명 | 예시 |
|------|------|------|
| `-t, --template` | 템플릿 타입 (1 또는 2) | `-t 1` |
| `--title` | 문서 제목 | `--title "사업 제안서"` |

### 공통 옵션

| 옵션 | 설명 | 기본값 | 예시 |
|------|------|--------|------|
| `--department` | 소속 부서 | OOOO부서 | `--department "디지털혁신부"` |
| `--author` | 작성자 | 작성자 | `--author "홍길동 팀장"` |
| `--date` | 작성일 (YYYY.MM.DD) | 오늘 | `--date "2026.03.11"` |

### 양식1 전용 옵션

| 옵션 | 설명 | 예시 |
|------|------|------|
| `--recipient` | 수신처 | `--recipient "기술혁신센터"` |

### 양식2 전용 옵션

| 옵션 | 설명 | 예시 |
|------|------|------|
| `--overview` | 개요/목적 | `--overview "AI 기반 업무 혁신"` |
| `--approaches` | 추진방안 (3개 이상) | `--approaches "방안1" "방안2" "방안3"` |
| `--budget` | 예산 (항목:금액) | `--budget "개발:10억원" "운영:5억원"` |
| `--notes` | 기타사항 (3개 이상) | `--notes "협업필요" "보안검토" "일정준수"` |

## 사용 예시

### 예시 1: 정식 보고서 (양식1)

```bash
python3 hwpx_generator.py \
    --template 1 \
    --title "2027년 클라우드 전환 사업 제안서" \
    --department "디지털혁신부" \
    --author "이영희 팀장" \
    --recipient "기술혁신센터"
```

**생성 파일**: `output/2027년클라우드전환사업제안서_20260311.hwpx`

### 예시 2: 신사업 보고서 (양식2)

```bash
python3 hwpx_generator.py \
    --template 2 \
    --title "AI 챗봇 시스템 도입 보고서" \
    --department "IT혁신부서" \
    --author "김철수 부장" \
    --overview "최신 AI 기술을 활용한 고객 서비스 혁신 및 업무 효율화" \
    --approaches "AI 모델 선정 및 검증" "시스템 아키텍처 설계 및 구축" "테스트 및 배포 전략 수립" \
    --budget "AI모델구매:10억원" "시스템개발:30억원" "운영및유지보수:5억원" \
    --notes "AI팀과 협업 필요" "보안 검토 필수" "2026년 12월 오픈 목표"
```

**생성 파일**: `output/AI챗봇시스템도입보고서_20260311.hwpx`

### 예시 3: 1분기 실적 보고 (양식2)

```bash
python3 hwpx_generator.py \
    --template 2 \
    --title "2026년 1분기 사업 실적 보고" \
    --department "경영관리팀" \
    --author "박영수 차장" \
    --overview "2026년 1분기 주요 사업 추진 현황 및 성과 보고" \
    --approaches "매출 목표 달성 현황" "신규 고객 확보 현황" "프로젝트 진행 현황"
```

### 일괄 예시 실행

```bash
chmod +x examples.sh
./examples.sh
```

## 양식 선택 가이드

### 양식1을 선택해야 하는 경우

✅ 공공기관, 대학교, 대기업 공식 문서
✅ 제안 요청서, 회계감사 보고서, 평가 문서
✅ 10페이지 이상의 상세 문서
✅ 목차와 섹션 구조가 필요한 경우
✅ 정형화된 양식이 필요한 경우

**구조**: 표지 + 목차 + 6개 섹션 + 별지

### 양식2를 선택해야 하는 경우

✅ 신사업 제안서, 내부 보고서
✅ 프로젝트 요약, 진행 보고
✅ 1-2페이지 요약 문서
✅ 빠른 의사결정을 위한 간략한 보고서
✅ 개요 → 방안 → 예산 중심의 구조

**구조**: 1페이지 요약 + 붙임

## 문제 해결

### Q: 파일이 생성되지 않습니다
```bash
# 템플릿 파일 확인
ls -la assets/

# 출력 디렉토리 확인
ls -la output/
```

### Q: 파일이 열리지 않습니다
- 한컴오피스 2014 이상 필요
- 또는 한컴오피스 뷰어 설치

### Q: 한글이 깨집니다
- 인코딩 문제: UTF-8 확인
- Python 3.6 이상 사용

### Q: 내용이 제대로 치환되지 않습니다
- 특수문자 사용 시 따옴표로 감싸기
- 예: `--title "2026년 \"특별\" 프로젝트"`

## 고급 사용법

### Python 코드에서 직접 사용

```python
from hwpx_generator import HwpxGenerator

# 생성기 초기화
generator = HwpxGenerator(template_type=2)

# 문서 생성
output_path = generator.generate(
    title='AI 챗봇 시스템 도입 보고서',
    department='IT혁신부서',
    author='김철수 부장',
    overview='최신 AI 기술을 활용한 고객 서비스 개선',
    approaches=['AI 모델 선정', '시스템 구축', '테스트 및 배포'],
    budget={'모델 구매': '10억원', '개발비': '30억원'}
)

print(f"생성된 파일: {output_path}")
```

### 배치 생성

```python
# 여러 문서 일괄 생성
documents = [
    {'title': '1분기 보고', 'department': '영업1팀'},
    {'title': '2분기 보고', 'department': '영업2팀'},
    {'title': '3분기 보고', 'department': '영업3팀'},
]

generator = HwpxGenerator(template_type=2)

for doc in documents:
    generator.generate(**doc)
```
