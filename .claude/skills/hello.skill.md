# hello

간단한 인사 스킬

## 설명

이 스킬은 사용자에게 인사합니다.

## Triggers

hello, 안녕, greeting, 인사

## 사용법

사용자가 "안녕"이라고 하면:
1. 현재 시간 확인
2. 적절한 인사말 출력
3. 오늘의 팁 제공

## 예시

```
사용자: "안녕하세요"
Claude: "안녕하세요! 좋은 아침입니다. ☀️
         오늘의 팁: 커피 한 잔과 함께 시작하세요!"
```

## 구현

- 아침(6-12시): "좋은 아침입니다 ☀️"
- 점심(12-18시): "좋은 오후입니다 🌤️"
- 저녁(18-22시): "좋은 저녁입니다 🌙"
- 밤(22-6시): "늦은 시간이네요 🌃"
