# 기자재 관리 시스템 - 팀원용 설치 가이드

## 🎯 목표
바탕화면 아이콘 하나로 모든 것 해결!

---

## 📋 설치 순서 (최초 1회만)

### 1단계: WSL 설치 확인

**Windows + X** → **Windows PowerShell (관리자)**

```powershell
wsl --version
```

**결과에 따라:**

✅ **버전 표시됨** → 2단계로 이동

❌ **오류 발생** → WSL 설치 필요:
```powershell
wsl --install -d Ubuntu-24.04
```
→ 재부팅 → Ubuntu 초기 설정 (사용자명/비밀번호)

---

### 2단계: 프로젝트 파일 복사

**개발자가 제공한 폴더를 복사**:

```
equipment_manager 폴더 전체를 다음 위치에 복사:
\\wsl.localhost\Ubuntu-24.04\home\[사용자명]\
```

**예시**:
```
\\wsl.localhost\Ubuntu-24.04\home\teamuser\equipment_manager
```

---

### 3단계: 바탕화면 아이콘 생성

**탐색기에서**:
1. 복사한 `equipment_manager` 폴더로 이동
2. `create-desktop-icon.vbs` 파일 더블클릭
3. "바로가기 생성 완료" 메시지 확인
4. 바탕화면에 **"기자재 관리"** 아이콘 생성 확인

---

## 🚀 사용 방법 (매일)

### 시작하기
```
바탕화면 "기자재 관리" 아이콘 더블클릭
→ 3초 대기
→ 브라우저 자동 실행
→ 로그인 후 사용!
```

### 로그인 정보
- **주소**: http://localhost:8502
- **아이디**: 부서별 계정 (관리자에게 문의)
- **비밀번호**: 관리자에게 문의

---

## ⚠️ 주의사항

### ✅ 해야 할 것
- 바탕화면 아이콘으로 실행
- 브라우저만 닫고 싶으면 브라우저만 닫기
- 퇴근 시 서버 종료 (검은 창 닫기)

### ❌ 하지 말아야 할 것
- 검은 창(CMD) 닫으면 서버도 종료됨
- WSL Ubuntu 삭제하지 말 것
- equipment_manager 폴더 이동/삭제 금지

---

## 🔧 문제 해결

### "이 페이지에 연결할 수 없습니다"
→ 서버가 실행되지 않았습니다
→ 바탕화면 아이콘 다시 더블클릭

### "wsl 명령을 찾을 수 없습니다"
→ WSL이 설치되지 않았습니다
→ 1단계부터 다시 진행

### 기타 문제
→ 개발자(담당자)에게 문의
