# 🏫 신한대학교 교육용 기자재 관리 시스템

신한대학교 6개 단과대학 37개 학과의 교육용 기자재를 체계적으로 관리하는 웹 기반 시스템입니다.

## 📋 프로젝트 개요

- **기술 스택**: Python 3.x, Streamlit, SQLite, bcrypt, openpyxl
- **개발 버전**: v1.0 (Phase 1 완료)
- **최종 업데이트**: 2026-03-25

## ✨ 주요 기능

### 🔐 인증 및 권한 관리
- bcrypt 암호화 적용
- 30분 자동 로그아웃
- 권한별 접근 제어 (admin, college)
- 7개 초기 계정 제공

### 📝 기자재 등록
- **품목코드**: 시설팀 부여 코드 수동 입력 (필수)
- **자산코드**: 학교 자산관리 코드 (선택)
- **예산부서**: 예산 집행 부서명 (선택)
- 품목명, 규격, 학과, 카테고리, 수량, 단가
- 구매일, 입고일, 납품업체, 보관위치
- 내용연수, 폐기예정일 자동 계산
- 사진 업로드 (최대 5장, 10MB)
- 필수항목 실시간 검증

### 🔍 기자재 조회 및 검색
- 계층형 필터 (단과대학 > 학과 > 카테고리)
- 입고일 범위 검색
- 키워드 검색 (품목명, 규격)
- 요약 통계 (건수, 수량, 총액)
- 사진 썸네일 표시
- 수리 이력 카운트 표시
- 16개 컬럼 목록 (자산코드, 예산부서 포함)

### ✏️ 기자재 수정 및 삭제
- 기존 데이터 로드 및 수정
- 사진 관리 (추가/삭제)
- 소프트 삭제 (is_deleted=1)
- 2단계 확인 절차

### 🖨️ 인쇄 기능 (3가지 양식)
- **📋 리스트 인쇄**: A4 가로, 14개 컬럼, 목록형
- **📇 카드 인쇄**: A4 세로, 상세 정보, 카드형
- **📄 상세 전체**: 전체 항목 출력
- 자산코드, 예산부서 필드 포함
- 인쇄용 최적화 CSS 적용

### 📊 엑셀 기능
- **다운로드**: 21개 컬럼 엑셀 파일 생성
- **템플릿**: 양식 다운로드 제공
- **일괄 등록**: 엑셀 파일로 대량 등록
- **미리보기**: 등록 전 데이터 확인
- 자산코드, 예산부서 컬럼 포함

### 🔧 수리·점검 이력
- 수리 유형 (수리, 정기점검, 부품교체, 보증수리, 자체수리)
- 수리일, 내용, 업체, 비용
- 다음 점검일 관리
- 기자재별 이력 조회

### 🏷️ 카테고리 관리
- 품목 마스터 관리
- 자산구분 관리
- 카테고리 관리

## 🚀 설치 및 실행


## 🗄️ 데이터베이스 스키마

### 주요 테이블

#### departments (학과 마스터)
- 37개 학과 정보
- `dept_code` (PK), `dept_name`, `college`, `college_code`

#### users (사용자 계정)
- bcrypt 암호화
- `username` (Unique), `password`, `college_code`, `role`

#### equipment (기자재 메인) - 26개 컬럼
- `item_code` (품목코드, Unique, 시설팀 부여)
- `asset_code` (자산코드, 선택)
- `budget_dept` (예산부서, 선택)
- `item_name` (품목명)
- `category`, `spec`, `dept_code`, `college_code`
- `quantity`, `unit_price`, `total_price`
- `purchase_date`, `arrival_date`, `vendor`
- `asset_type`, `location`, `useful_life`, `disposal_date`
- `registrant_name`, `registrant_id`
- `is_deleted` (소프트 삭제)

#### equipment_photos (기자재 사진)
- `equipment_id` (FK), `file_name`, `file_path`
- `sort_order` (1번이 대표사진)

#### equipment_repairs (수리·점검 이력)
- `equipment_id` (FK), `repair_type`, `repair_date`
- `description`, `vendor`, `cost`, `status`

#### categories (카테고리 마스터)
- 실험기구, 전자기기, 의료기기, 공구/장비 등

#### asset_types (자산구분 마스터)
- 교육용기자재 등

#### equipment_items (품목 마스터)
- 노트북 컴퓨터, 프로젝터, 프린터 등 공통 품목

## 🔒 보안 기능

- ✅ bcrypt 비밀번호 해싱 (salt rounds)
- ✅ 세션 타임아웃 (30분)
- ✅ 권한별 접근 제어 (RBAC)
- ✅ 소프트 삭제 (데이터 복구 가능)
- ✅ 파일 업로드 검증 (확장자, 용량, MIME 타입)
- ✅ SQL Injection 방지 (Parameterized Query)
- ✅ XSS 방지 (Streamlit 기본 제공)
