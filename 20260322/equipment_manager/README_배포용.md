# 기자재 관리 시스템 - 팀원 배포 가이드

## 📦 배포 파일 목록

팀원들에게 다음 폴더 전체를 전달하세요:
```
equipment_manager/
├── EquipmentManager.bat          ← 메인 실행 파일
├── start.sh                       ← 서버 시작 스크립트
├── stop.sh                        ← 서버 종료 스크립트
├── main.py                        ← 메인 앱
├── auth.py                        ← 인증 모듈
├── database.py                    ← 데이터베이스
├── pages/                         ← 페이지들
├── data/                          ← 데이터 폴더
├── Setup-EquipmentManager.ps1     ← 자동 설치 스크립트
├── create-desktop-icon.vbs        ← 수동 아이콘 생성
└── INSTALL_FOR_TEAM.md           ← 팀원용 가이드
```

---

## 🚀 팀원 설치 방법 (2가지 선택)

### 방법 A: 자동 설치 (추천) ⭐

**1단계**: 폴더 받기
- equipment_manager 폴더 전체를 USB/공유폴더에서 복사

**2단계**: WSL 경로에 복사
```
\\wsl.localhost\Ubuntu-24.04\home\[본인이름]\
```
→ equipment_manager 폴더를 여기에 붙여넣기

**3단계**: 자동 설치 실행
- `Setup-EquipmentManager.ps1` 우클릭
- **PowerShell에서 실행** 클릭
- 안내에 따라 진행

**완료!** 바탕화면에 "기자재 관리" 아이콘 생성

---

### 방법 B: 수동 설치

**1단계**: WSL 설치 확인
```powershell
wsl --version
```
- 없으면: `wsl --install -d Ubuntu-24.04`

**2단계**: 폴더 복사
```
equipment_manager → \\wsl.localhost\Ubuntu-24.04\home\[이름]\
```

**3단계**: 아이콘 생성
- `create-desktop-icon.vbs` 더블클릭

---

## 💡 팀원 사용 방법

### 매일 사용
```
바탕화면 "기자재 관리" 더블클릭
→ 브라우저 자동 실행
→ 로그인 후 사용
```

### 종료
```
검은 창(CMD)에서 아무 키나 누르기
또는 창 닫기
```

## 🔐 계정 관리

### 기본 관리자 계정
- ID: `admin`
- PW: `admin123`

### 팀원 계정 생성
1. 관리자로 로그인
2. **관리자 메뉴** → **사용자 관리**
3. 신규 사용자 등록

### 부서별 계정 권한
- **admin**: 전체 접근
- **부서 계정**: 해당 부서만 접근

## ⚠️ 주의사항

### 데이터 백업
각 팀원 PC에서 개별 실행 시:
```
equipment_manager/data/equipment.db
```
→ 주기적으로 백업 필요

### 공유 사용 시
한 대의 서버 PC에서만 실행하고
나머지는 네트워크로 접속:
```
http://[서버PC-IP]:8502
```

## 🔧 문제 해결

### "이 페이지에 연결할 수 없습니다"
1. 바탕화면 아이콘 다시 실행
2. 검은 창이 열리는지 확인
3. 3초 대기 후 브라우저 새로고침

### "wsl 명령을 찾을 수 없습니다"
→ WSL 설치 필요
```powershell
wsl --install -d Ubuntu-24.04
```
→ 재부팅 후 다시 시도

### 기타 문제
→ 시스템 관리자에게 문의
