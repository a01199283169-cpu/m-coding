# hwpx-bogosu

> 한글(hwpx) 보고서 자동 생성 스킬

한글(hwpx) 문서를 자동으로 생성하는 Claude Code 스킬입니다.
두 가지 표준 양식을 지원하여 다양한 보고서 작성을 간편하게 처리할 수 있습니다.

## 특징

- ✅ **두 가지 양식**: 정식 보고서 / 요약 보고서
- ✅ **자동 생성**: Python 스크립트로 hwpx 파일 자동 생성
- ✅ **구조 보존**: ZIP 압축 구조 완벽 유지
- ✅ **한컴 호환**: 한컴오피스 2014+ 지원
- ✅ **간편 사용**: 자연어 요청만으로 문서 생성

## 빠른 시작

### 1. 설치

```bash
# 스킬 패키지를 Claude Code에 등록
# 1) 이 폴더를 원하는 위치로 복사
# 2) Claude Code에서 스킬 경로 등록
```

### 2. 기본 사용

**자연어 요청:**
```
"영업팀 2026년 1분기 실적 보고서 만들어줘. 작성자는 홍길동."
```

**명령어:**
```bash
python3 hwpx_generator.py -t 2 \
    --title "2026년 1분기 실적 보고서" \
    --department "영업팀" \
    --author "홍길동" \
    --overview "2026년 1분기 주요 사업 추진 현황 및 성과"
```

**출력:**
```
✅ hwpx 파일 생성 완료: output/2026년 1분기 실적 보고서_20260311.hwpx
```

### 3. 고급 사용

**신사업 보고서 (상세 옵션):**

## 기술 스펙

### hwpx 파일 포맷

- **형식**: ZIP 기반 XML 문서
- **호환**: 한컴오피스 2014 이상
- **구조**:
  - `mimetype` (첫 번째 파일, uncompressed)
  - `version.xml` (문서 버전 정보)
  - `Contents/` (본문 내용)
  - `Preview/` (미리보기 데이터)
  - 기타 메타데이터

### ZIP 구조 요구사항

1. **mimetype 파일**
   - 반드시 첫 번째 파일
   - 압축 안 됨 (ZIP_STORED, compress_type=0)
   - Extra field 없음

2. **파일 순서**
   - 원본 템플릿과 동일한 순서 유지
   - 파일 순서가 바뀌면 한컴오피스에서 열리지 않을 수 있음

3. **압축 타입**
   - 각 파일의 원본 압축 방식 보존
   - compress_map 사용하여 관리

## 문제 해결

### Q1. 생성된 파일이 열리지 않아요

**확인 사항:**
1. 한컴오피스 2014 이상인지 확인
2. 파일 크기가 0이 아닌지 확인
3. mimetype 파일이 첫 번째이고 uncompressed인지 확인

**해결 방법:**
```bash
# 파일 구조 확인
python3 -c "
import zipfile
with zipfile.ZipFile('output/파일명.hwpx', 'r') as z:
    for info in z.infolist()[:3]:
        print(f'{info.filename}: {info.compress_type}')
"
# 첫 번째 줄이 'mimetype: 0' 이어야 함
```

### Q2. 특수문자가 포함된 제목을 사용하고 싶어요

**해결 방법:**
```bash
# 따옴표로 감싸기
--title "2026년 \"특별\" 프로젝트 제안서"
```

### Q3. 여러 개를 한 번에 만들고 싶어요

**해결 방법:**
```bash
# 배치 스크립트 작성
for dept in "영업팀" "마케팅팀" "개발팀"; do
    python3 hwpx_generator.py -t 2 \
        --title "2026년 1분기 실적 보고서" \
        --department "$dept" \
        --author "홍길동"
done
```
